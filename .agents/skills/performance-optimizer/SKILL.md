# Performance Optimizer

This skill enables the agent to analyze system performance and recommend optimizations.

---

# When to Use This Skill

Use this skill when the user:

- asks how to make an app faster
- reports slow systems
- wants to optimize queries
- needs performance tuning
- wants scalability improvements

---

# Performance Analysis

The agent should analyze:

- CPU usage
- memory usage
- database performance
- network latency
- rendering performance

---

# Optimization Strategies

The agent should recommend:

- caching
- query optimization
- indexing
- lazy loading
- code optimization

---

# Observability

The agent should suggest tools such as:

- Prometheus
- Grafana
- ELK stack
- APM tools
